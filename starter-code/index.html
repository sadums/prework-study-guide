HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Meta tags are used to tell the browser information about your webpage, but are not visible to users.
  • Header tags should be used in order of h1 to h6, with h1 being the largest.

CSS

  • Margin indicates space on the outside of the border.
  • Padding indicates space on the inside of the border.
  • CSS can be written internal/inline with the HTML file, however it is far more popular to have a seperate CSS file or external.
  • When a CSS sheet is external it must be linked to the HTML page with a link tag.

Git

  • "git status" will show which branch you are currently on.
  • "git checkout -b branch name" will create a new branch.
  • The git command must precede every Git statement.
  • "git add -A" to add all changes to the staging area.
  • 'git commit -m "commit name" ' will commit changes in the staging area.
  • "git pull origin main" will check if local branch is in sync with main branch.
  • "git push origin 'branch-name'" will push changes to the remote branch.
